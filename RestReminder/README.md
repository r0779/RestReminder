# 休息提醒 (Rest Reminder)

一个简洁的Android应用，帮助用户定时休息，保护健康。

## 功能特点

- ✅ **定时提醒**：可自定义提醒间隔时间（默认30分钟）
- ✅ **多种提醒方式**：弹窗通知、系统提示音、振动提醒
- ✅ **后台运行**：应用在后台时仍能准确触发提醒
- ✅ **暂停/恢复**：支持临时暂停和恢复提醒功能
- ✅ **简洁界面**：直观易用的用户界面

## 使用说明

1. **开启提醒**：打开"开启提醒"开关
2. **设置间隔**：输入想要的提醒间隔时间（分钟）
3. **保存设置**：点击"保存设置"按钮
4. **暂停/恢复**：使用暂停和恢复按钮控制提醒

## 技术特点

- 使用 AlarmManager 实现精确的定时提醒
- 支持系统通知栏显示
- 兼容 Android 8.0 及以上系统
- 使用 SharedPreferences 保存用户设置

## 权限说明

- `POST_NOTIFICATIONS`：发送通知权限
- `SCHEDULE_EXACT_ALARM`：设置精确闹钟权限（Android 12+）

## 系统要求

- Android 8.0 (API 26) 或更高版本
