# 本地编译 APK 指南

## 方法一：使用 Android Studio 构建（推荐）

### 步骤：
1. **打开 Android Studio**
2. **打开项目**
   - File → Open
   - 选择 `C:\Users\roro\Desktop\RestReminder` 文件夹
   - 等待 Gradle 同步完成（首次需要几分钟）

3. **构建 APK**
   - 菜单栏选择：Build → Build Bundle(s) / APK(s) → Build APK(s)
   - 或者按快捷键：`Ctrl + F9`

4. **查看构建结果**
   - 构建完成后，右下角会弹出通知
   - 点击 "locate" 查看生成的 APK 文件
   - APK 位置：`app\build\outputs\apk\debug\app-debug.apk`

---

## 方法二：使用命令行编译

### 前提条件：
1. 安装 Android Studio
2. 项目首次需要在 Android Studio 中打开并同步 Gradle

### 步骤：
1. 在项目根目录下运行 `build-local.bat` 脚本
2. 等待编译完成
3. 编译成功后，APK 会自动在资源管理器中打开

---

## 注意事项

### 首次编译：
- 首次打开项目时，Gradle 会自动下载依赖，可能需要 5-10 分钟
- 请确保网络连接正常
- 如果 Gradle 同步失败，尝试：
  - File → Invalidate Caches → Invalidate and Restart
  - 或者重新打开项目

### 编译错误处理：
1. **SDK 位置未设置**
   - File → Project Structure → SDK Location
   - 设置 Android SDK 路径（通常在 `C:\Users\roro\AppData\Local\Android\Sdk`）

2. **Gradle 版本问题**
   - File → Settings → Build, Execution, Deployment → Gradle
   - 选择 "Use Gradle wrapper" 并指定版本

3. **依赖下载失败**
   - 检查网络连接
   - 配置 Gradle 代理（如果需要）

---

## 快速测试

编译完成后，可以通过以下方式安装到手机：

1. **通过 USB**
   - 手机开启"开发者选项"和"USB调试"
   - 连接电脑
   - 在 Android Studio 中点击 Run 按钮（绿色三角形）

2. **手动安装**
   - 将 `app-debug.apk` 复制到手机
   - 在手机上安装（需允许未知来源）
